# SaaS AI Disruption Index

Interactive scatter plot visualization scoring SaaS companies on their vulnerability to AI disruption.

## Disruption Score

A composite score (1-10) derived from three factors:

- **Domain Expertise Overlap (45%)** - Can end users build AI alternatives themselves?
- **Small Business Exposure (30%)** - SMB-heavy companies face higher switching risk
- **Standard-Setter Defensibility (25%, inverted)** - Category standards have stronger moats

## Usage

Open `index.html` in a browser, or deploy as a static site (GitHub Pages compatible).

## Data

`data/companies.json` contains 31 SaaS companies with scores and methodology documentation.
